# js-executor

To install dependencies:

```bash
bun install
```

To run ( for test function is working ):

```bash
bun run dev --width 400 --height 300 --concurrency 1 < test.json
```

Build executable file: <https://bun.com/docs/bundler/executables>

```bash
bun run build:dev
bun run build:dev-win
bun run build:prod:x64
bun run build:prod:arm64
```

This project was created using `bun init` in bun v1.3.3. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
